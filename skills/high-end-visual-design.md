# Premium Visual Design for Zoggy Studios

## Principles

- Every project page deserves its own visual identity within the studio brand.
- Use the shared layout primitives (page shell, cards, buttons) but vary colors, spacing, and imagery.
- Dark fantasy projects should feel ancient and corrupted.
- Technology/tool pages should feel precise and developer-oriented.
- Experimental/creative pages should feel playful and loose.

## Creative Variance for Projects

When building project pages, rotate through these approaches:

1. **Asymmetric hero** — text left, image right at 60/40 split
2. **Full-bleed image hero** — atmospheric background with overlaid text
3. **Split content** — alternating image and text blocks down the page
4. **Card grid showcase** — bento-style grid of feature cards
5. **Timeline/checklist** — horizontal or vertical progress visualization

Do not use the same layout for two consecutive project pages.
