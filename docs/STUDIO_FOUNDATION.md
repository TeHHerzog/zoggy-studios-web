# Zoggy Studios Foundation

## Purpose

This document captures the research-backed direction for building Zoggy Studios as the primary public site before expanding the Dark Fantasy MMO site.

## Research Summary

Creator and studio website guidance consistently points to a small set of essentials:

- A homepage that quickly explains who you are, what you make, and where visitors should go next.
- A curated work or projects page instead of a large fake portfolio.
- A clear About page with honest context and positioning.
- A direct contact path, usually email-first for early-stage studios.
- Updates, blog, newsletter, media kit, and press pages only when they can be maintained or backed by real assets.

Indie game press guidance also recommends separating the studio layer from individual game pages. A studio page explains who made the work and accumulates history across projects; a game page or press kit explains one release when screenshots, descriptions, factsheets, and contact material are ready.

Author and writing-site guidance overlaps with this: keep an About page, project/book pages when real works exist, contact information, and optionally media or newsletter material later.

## Direction Chosen

Build `games.herzogit.com` as the Zoggy Studios home base first.

Current routes:

- Home
- Projects
- About
- Updates
- Contact
- Privacy
- Terms

Dark Fantasy MMO remains a restrained project card and separate placeholder site until more public game material is ready.

## Content Boundaries

Do not fake:

- shipped games
- press coverage
- testimonials
- team size
- awards
- public release dates
- business/legal maturity
- finished books or scripts
- public MMO systems

Do show:

- current focus
- future shelves
- honest solo-led status
- broad creative direction
- contact path
- security boundary

## Later Additions

Consider these only when there is real material to support them:

- media or press kit page
- newsletter signup
- individual project pages
- app landing pages
- book pages
- script request workflow
- social links
- store links
- formal legal policies
- business entity details
